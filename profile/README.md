## 🏠 What is Nexo?

Nexo is a growing collection of personal PWAs that install on your phone's home screen, share one login, and live on one server. No subscriptions. No tracking. Just useful tools built exactly the way I want them.

## 🚀 Live apps

| | App | What it does |
|---|---|---|
| 💰 | [**Finance**](https://finance.krieger2501.de) | Expense, income & recurring commitment tracker |
| 🔐 | [**Auth**](https://auth.krieger2501.de) | Central OIDC login — Google, GitHub & Discord |
| 🛠️ | [**Admin**](https://admin.krieger2501.de) | Container management & user access control |
