# Security Policy

## Reporting a vulnerability

Please **do not** open a public issue for security vulnerabilities.

Report via [GitHub's private vulnerability reporting](https://github.com/nexo-suite/nexo/security/advisories/new) — this keeps the details confidential until a fix is in place.

I aim to respond within 72 hours. This is a hobby project with a small user base, but I take security seriously and will fix and credit promptly.

## Scope

- The apps running at `krieger2501.de` and their subdomains
- The `nexo-suite/nexo` codebase and Docker images

## Out of scope

- Attacks requiring physical access to the VPS
- Social engineering of users
- Denial of service against a €5/mo server (already assumed)
